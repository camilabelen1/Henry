AGREGAR COSAS

Primero debe agregar y enviar el archivo png a su repositorio local. Luego, envíe las actualizaciones al servidor remoto.

git add xxx.png
git commit -m 'add png file'
git remote add origin https://github.com/xxx.git
git push -u origin master
